-) Project Overview

This project presents an interactive Power BI dashboard designed to analyze mobile phone sales data and deliver
actionable business insights. The dashboard helps understand sales performance across cities, brands,
mobile models, and payment methods.

-) Objectives

Track overall sales performance using key KPIs

Analyze monthly sales and quantity trends

Identify top-selling mobile models and brands

Understand city-wise sales distribution

Analyze customer behavior through ratings

Study transaction distribution by payment methods

-) Dashboard Highlights

KPIs: Total Sales, Total Transactions, Average Price, Quantity Sold

Time Analysis: Monthly sales and quantity trends

Geographical View: City-wise sales distribution

Product Analysis: Sales by mobile model

Customer Insights: Ratings analysis

Payment Analysis: Transactions by payment method

Interactivity: Filters for month, brand, model, and payment method

-) Tools & Technologies

Power BI

DAX

Data Modeling

Interactive Visualizations
